# DESCRIPTION

This project is a **Decentralized Finance (DeFi) Application** built using **Solidity smart contracts** and a **React.js frontend**, enabling users to deposit, borrow, and repay cryptocurrency in a trustless environment.

The system simulates a lending pool with a **dynamic interest rate mechanism** based on liquidity conditions in the smart contract.


# Objectives  

- Develop a decentralized lending and borrowing system  
- Implement smart contracts using Solidity  
- Create a user-friendly frontend using React.js  
- Integrate blockchain using ethers.js and MetaMask  
- Deploy and test on Ethereum Sepolia test network  


#  Key Features  

- Deposit ETH into the smart contract  
- Borrow ETH based on available liquidity  
- Repay borrowed funds securely  
- Dynamic interest rate adjustment based on contract balance  
- Wallet connection using MetaMask  
- Transparent on-chain transactions  

---

# Smart Contract Functionality  

- Lending pool management  
- Borrowing mechanism with liquidity check  
- User balance and debt tracking  
- Dynamic interest rate calculation  
- Secure ETH transfer between users and contract  



#  Technologies Used  

- Solidity (Smart Contracts)  
- React.js (Frontend)  
- Ethers.js (Blockchain Integration)  
- MetaMask Wallet  
- Ethereum Sepolia Test Network
