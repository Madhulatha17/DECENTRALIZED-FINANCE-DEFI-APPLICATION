#  DeFi DApp Deployment Steps

## Tools Required
- Remix IDE
- MetaMask Wallet
- Sepolia Testnet
- Node.js + React



##  Smart Contract Deployment

1. Open Remix IDE
2. Paste DeFiLending.sol
3. Compile (Solidity 0.8.20+)
4. Connect MetaMask
5. Select Sepolia Test Network
6. Deploy contract
7. Copy contract address


##  Frontend Setup

```bash
cd frontend
npm install
npm start
